# BabyCam Monitor Server

## Hardware:
- Raspberry Pi 3
- Pi NOIR Camera V2

## Requirements
- Python 3
- Make

## Install
Type inside the project directory:
```bash
$ make install
```
## Uninstall
Type inside the project directory:
```bash
$ make uninstall
```
## Run
Type inside any directory:
```bash
$ babystream
```
